Instruções

Para disponibilizar as páginas estáticas: Instalar o http-server de forma global caso ainda não tenha instalado. Para isso digite npm install --global http-server no terminal. Em seguida abrir o terminal na pasta raiz do projeto, onde está localizado o arquivo index.html, e digitar http-server -c-1 para iniciar o servidor(por padrão na porta 8080). A partir disso, as páginas poderão ser acessadas via localhost. A opção -c-1 informa ao servidor que não guarde arquivos em cache

Para disponibilizar um servidor para ficar aguardando por submissões de formulários de cadastro: Através de um terminal, acessar a pasta raiz "formularios", onde está localizado o arquivo server.js, que contém a lógica do servidor, e que será disponibilizado através do web framework express. Use o comando npm install para que as dependências do express e do body-parser, que estão descritas no package.json, sejam instaladas. digitar node server.js para iniciar o servidor na porta 3003
